Flock XR project embed

This page embeds a remote Fidget spinner.flock project in player-only mode.

Optional preset via URL: ?size=compact, ?size=default, ?size=tall, or ?size=full.

Open in full editor: open project on flipcomputing.github.io/flock

Embed code snippet

Copy this snippet into your own web page:

<iframe
  title="Flock XR embedded project player"
  allow="fullscreen; xr-spatial-tracking; accelerometer; gyroscope"
  src="https://flipcomputing.github.io/flock/?embed=true&project=YOUR_RAW_FLOCK_URL"
></iframe>

Hosting your project on GitHub

Upload your .flock file to a GitHub repository, then copy the file’s Raw URL and use that as the project= value in the embed link.
