Flock XR embed example

Use this URL directly in an iframe to load a remote Fidget spinner.flock project (served with CORS headers) in player-only mode.

It uses ?project= plus ?embed=true on the main app URL. 

<iframe
  title="Flock XR embedded project"
  allow="fullscreen; xr-spatial-tracking; accelerometer; gyroscope"
  src="./index.html?embed=true&project=https%3A%2F%2Fraw.githubusercontent.com%2Fflipcomputing%2Fflockxr-projects%2Frefs%2Fheads%2Fmain%2FFidget%2520spinner.flock"
></iframe>
