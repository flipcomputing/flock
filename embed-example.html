Flock XR embed example

This page uses a plain iframe embed. Set the size URL parameter on the iframe source (for example size=default or size=compact) along with controls=playstop.

Copy/paste embed snippet

<iframe
  title="Flock XR embedded project player"
  width="800"
  height="450"
  style="border: 0"
  allow="fullscreen; xr-spatial-tracking; accelerometer; gyroscope"
  src="https://your-host/flock/index.html?embed=true&size=default&controls=playstop&project=YOUR_RAW_FLOCK_URL"
></iframe>
